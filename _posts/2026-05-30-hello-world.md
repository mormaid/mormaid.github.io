Hello!

I'm kicking off a new blog to keep track of my security notes on technical fundamentals, memory cheatsheets and learning logs, and probably serial posts on any build projects I have on the go. 

I'm currently building a home media inventory app (MIA), to keep track of my library of various physical media (books, vinyls, blu-rays, whatever) as I wean myself off Big Tech and multiple crappy streaming services pushing ads with nothing I want to watch.

My other project is a little cyberdeck built on a Raspberry Pi 5, which will be a cute and portable writing station (though I'm not sure how portable, since I went for the HAT/SSD and active cooling/fan unit).  

General goal of the blog is to document what I’m learning for my sake, keep track of any cheatsheets/notes I put together that explain things in the only way I can understand (hello IPv4 subnetting and ElGamal encryption), and keep the whole site small enough that I understand how it all hangs together. I'm over digging through other people's uncommented code and trying to work out which file in the repo does what.
